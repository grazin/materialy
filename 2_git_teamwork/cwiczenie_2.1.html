  1. Captain Kirk
  2. Mr. Spock

Zdecydujcie kto będzie Kirkiem, kto Spockiem.

Użyjcie repozytorium z poprzedniego ćwiczenia kapitana Kirka.

Sklonuj repozytorium kapitana Kirka do nowego folderu (poza swoim własnym repozytorium). Przejdź do tego repozytorium i dla pewności sprawdź status. 

$ git clone https://github.com/grazin/repo-cpt-kirk.git
odpowiedź:
Cloning into 'repo-cpt-kirk'...

$ cd repo-cpt-kirk

$ git status

Stwórz brancz feature-kirk-1.

$ git checkout -b feature-kirk-1

Stwórz nowy plik o nazwie stepy.akermanskie.txt i wprowadź następującą treść:

Wpłynąłem na suchego przestwór oceanu,
Wóz nurza się w zieloność i jak łódka brodzi,
Śród fali łąk szumiących, śród kwiatów powodzi,
Omijam koralowe ostrowy burzanu.

Zapisz zmiany, zakomituj i pusznij zmiany na brancz zdalny. Przełącznik -u jest konieczny, ponieważ to nowy bracz, którego nie ma jeszcze na zdalnym repozytorium. 

$ git add .

$ git commit -m "cpt. Kirk stepy akermanskie"

$ git push -u origin feature-kirk-1

Stwórz brancz feature-spock-1.

$ git checkout -b feature-spock-1

Stwórz nowy plik o nazwie w.malinowym.chruśniaku.txt i wprowadź następującą treść:

W malinowym chruśniaku, przed ciekawych wzrokiem
Zapodziani po głowy, przez długie godziny
Zrywaliśmy przybyłe tej nocy maliny.
Palce miałaś na oślep skrwawione ich sokiem.

Zapisz zmiany, zakomituj i pusznij zmiany na brancz zdalny. Przełącznik -u jest konieczny, ponieważ to nowy bracz, którego nie ma jeszcze na zdalnym repozytorium.

Przejdź na brancz Spocka feature-spock-1. Przejrzyj zmiany w dowolnym GUI.

$ git checkout feature-spock-1

Przejdź na brancz Kirka feature-kirk-1. Przejrzyj zmiany w dowolnym GUI.

$ git checkout feature-kirk-1

Jeśli Spock zatwierdził twoje zmiany (ustnie wyraził zadowolenie z Twoich zmian), wróc na brancz główny, żeby następnie swoje zmiany zmerdżować.

$ git checkout master

Jeśli Kirk zatwierdził twoje zmiany (ustnie wyraził zadowolenie z Twoich zmian), wróc na brancz główny, żeby następnie swoje zmiany zmerdżować.

$ git checkout master

Najpierw Kirk. Zmerdżuj swój brancz i pusznij.

$ git merge feature-kirk-1

Teraz Spock. Najpierw uaktualnij główny brancz poleceniem git pull.

$ git pull
